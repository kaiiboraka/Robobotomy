## Controls

### Gameplay

<!-- - Move left / right: <kbd>A</kbd> / <kbd>D</kbd> -->
- Move left / right: <kbd>A</kbd> / <kbd>D</kbd>
- Jump: <kbd>Space</kbd>
- Interact: <kbd>E</kbd>
- Climb up / down: <kbd>W</kbd> / <kbd>S</kbd>
- Open radial menu: <kbd>Tab</kbd>
- Throw limb: <kbd>R</kbd> or right-click
- Recall limb: <kbd>X</kbd>

### Shortcuts

- Quit game: <kbd>Alt</kbd> + <kbd>Q</kbd>
- Toggle Fullscreen: <kbd>F11</kbd> or <kbd>Alt</kbd> + <kbd>Enter</kbd>

### Debug

- Game speed up: <kbd>.</kbd>
- Game slow down: <kbd>,</kbd>
- Game speed reset: <kbd>/</kbd>
- Toggle mouse debug: <kbd>F1</kbd>
